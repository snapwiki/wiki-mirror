To access Dev Mode, shift + click the Snap! logo at the top left corner of the editor and click on "Switch to dev mode." Dev Mode allows the morphs to be inspected and edited. Everywhere you right-click, a morphic context menu appears. There are extra primitives that only appear in Dev Mode.
