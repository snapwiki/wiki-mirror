Beetle Blocks is a Snap! modification with the ability to render the output in 3D, and even share 3D projects.

Because of a problem with the cloud in the server of Beetle Blocks, all the project data were lost.[1] In Snap! 10, a library reminiscent of Beetle Blocks called 3D Beetle Geometry is added.
