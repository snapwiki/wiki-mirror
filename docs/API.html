Both Snap! and Snap! Cloud have an API. This page describes them.

Snap!

The Snap! API allows accessing Snap! from outside programs to start scripts, send and retrieve information. For example, using the API when embedding interactive projects in an iframe. Documentation for the Snap! API can be found here.

Snap! Cloud

The Snap! Cloud API allows accessing data from the Snap! community website. For example, getting a list of all published remixes from a project. Documentation for the Snap! Cloud API can be found here. Accessing these from Snap! projects used to be possible but this changed sometime in 2021.[1]
