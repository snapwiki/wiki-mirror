Motion is the first category of blocks, and it carries over 13 commands, and 3 reporters. These blocks are used for sprite movement.

BlockFunctionOptions
Commands (13)
move (10) stepsMoves the sprite forward (or backward) from the direction the sprite is at.
turn cw (15) degrees :: motion

turn ccw (15) degrees :: motion
Turns the sprite left or right by a certain amount of degrees.
point in direction (90 v) :: motionSets the sprite's direction.
  • Random: Picks a random angle (number from 0 to 359).
point towards [ v]Makes the sprite point toward something.
  • Random Position: Points at a random position.
  • Mouse Pointer: Points at the mouse pointer.
  • Center: Points at the center (0, 0).
go to x: (0) y: (0) :: motion

set x to (0) :: motion

set y to (0) :: motion
Sets the sprites position (x, y).
go to [ v] :: motionMakes the sprite go to the position of something.
  • Random Position: Goes to a random position.
  • Mouse Pointer: Goes to the mouse pointer.
  • Center: Goes to the center (0, 0).
glide (1) secs to x: (0) y: (0) :: motion"Glides" the sprite to the position, or go to the position smoothly. The higher amount of seconds makes the sprite go slower.
change x by (10) :: motion

change y by (10) :: motion
Changes the x or y of the sprite's position.
if on edge, bounce :: motionFlips/mirrors the sprite when the it's touching the edge (out of bounds).
Reporters and Predicates (3)
x position :: reporter motion
y position :: reporter motion
direction :: reporter motion
Returns the sprite's positions or direction.
