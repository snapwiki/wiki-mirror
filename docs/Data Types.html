In Snap!, data types define different formats of data.

Primitive Data Types

Basic Data Types

Number

Numbers are considered to be a base-10 set of digits. Extraneous zeroes (like 001 or 0.020) are trimmed. Numbers can be negative and either be an integer or have decimals.
