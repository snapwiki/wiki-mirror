Turbo Mode is feature introduced in Scratch and Snap! that diverts processing from display (such as the motion of sprites on the stage) to the running of user scripts. This will make scripts run faster than usual, and it is useful for complex projects. Turbo Mode is enabled by pressing the shift key when clicking the Green Flag. It can also be enabled in the project editor's settings.

Uses

Turbo mode is mainly used for speeding up computation-heavy projects.

Projects that might use turbo mode include, but are not limited to:

  • Drawing projects
  • Big mathematical operations
  • 3D engines
  • Pen projects
