The costumes list can be modified programmatically by using the following list stack blocks add [] to @list delete ( v) of @list insert [] at ( v) of @list replace item ( v) of @list with [] putting the (my [costumes V]) block in the list input.

You can also do the same thing with sounds, swap "costumes" with "sounds".

Example Uses

  • Copying costumes from one sprite to another
  • Adding costumes extracted from the web by (primitive [cst_load(url) V] [] @delInput @addInput)
  • Adding a costume built using the (new costume @list width ( v) height ( v))
  • Performing image actions (such as blur (see the Animations library), flip, stretch and skew[upcoming in 10.0]) and adding the modified version without drag-and-drop
