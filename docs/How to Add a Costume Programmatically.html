The costumes list can be modified programmatically by using the following list stack blocks <sb>add [] to @list::list</sb> <sb>delete ( v) of @list::list</sb> <sb>insert [] at ( v) of @list::list</sb> <sb>replace item ( v) of @list with []::list</sb> putting the <sb>(my [costumes v]::sensing)</sb> block in the list input.

You can also do the same thing with sounds, swap "costumes" with "sounds".

Example Uses

  • Copying costumes from one sprite to another
  • Adding costumes extracted from the web by <sb>(primitive [cst_load\(url\) v] [] @delInput @addInput::grey)</sb>
  • Adding a costume built using the <sb>(new costume @list width ( v) height ( v)::looks)</sb>
  • Performing image actions (such as blur (see the Animations library), flip, stretch and skew[upcoming in 10.0]) and adding the modified version without drag-and-drop
