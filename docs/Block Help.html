Block Help (also known as only Help or Help Screens) is a Snap! feature that tells the user what each of the primitive blocks do as the name suggests. Most of the block help screens were made by Nathan Dinsmore who was a major Snap! contributor. Many of the help screens were also inherited from Scratch 1.x.

Blocks That Do Not Have Block Help

The following blocks do not have block help:

  • Blocks which belong to libraries except for the throw and catch blocks in the Iteration, composition library.
  • (new costume @list width ( v) height ( v)
  • (new sound @list rate (44100 v) Hz)
  • paste on [ V]
  • send [ V] to [ V] :: control
  • (timer)
  • (index of [thing] in @list)
  • (append @list @list @delInput @verticalEllipsis @addInput)

Snap! developers have been asked to do something about this but so far no action has been taken.
