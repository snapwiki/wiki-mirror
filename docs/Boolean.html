A Boolean can be either true or false. Booleans can be used identify on/off states, binary values, etc..

Blocks

Blocks that maniuplate Booleans include (not including predicates or blocks with Boolean slots that are not centered around Booleans):

<<true>>: Reports a Boolean.

<not <>>: Reports the inverse Boolean (true reports false and false reports true).
