The file menu can be accessed by clicking on the piece of paper to the top-left of the Snap! editor. It contains options related to the current file or project.

Options

  • Notes: Opens a dialog to add or change the project notes.
  • New: Replaces the current project with the default project. Asks the user if they want to save the current project first.
  • Open: Opens a dialog to open a project. The user can also share, publish and delete projects from the dialog. There are three sections to choose projects from: Cloud (their own projects if logged in), Examples (example projects) or Computer (the user will be prompted to choose a Snap!-formatted XML from their computer).
  • Save: Saves the current project in the last used location. If the project wasn't saved, the user will be asked where to save. If the project was saved in the cloud, the current version of the project will override the last version. If the project was saved into the computer before, a separate one will be saved.
  • Save As: Like Save, but acts as if it was a new project.
  • Restore unsaved project: Tries to recover the last unsaved project. Appears if the previous project wasn't saved and the replacing project wasn't edited yet.
  • Import: Opens a file prompt to choose a XML that was exported from Snap!.
  • Export project
  • Export summary
  • Export pen trails[upcoming in 10.0]
  • Hide blocks: Opens a prompt to choose which blocks to hide. On the prompt, click on a checkbox (not a block as this will run it) to hide a block. Right-click on the background of the prompt to quickly hide a group of blocks.
  • New category
  • Remove a category
  • Generate puzzle: Automatically generates a Parsons problem from a sprite.
  • Scenes
  • New scene
  • Add scene
  • Libraries: Opens a menu to pick a library to import.
  • Costumes: Opens a gallery of pre-existing costumes.
  • Backgrounds: Like Costumes, but this appears instead of Costumes when the stage is selected.
  • Sounds: Opens a gallery of sounds.
  • Undelete sprites
