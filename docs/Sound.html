Sound is the third category of blocks, and it carries over 15 blocks, and 5 reporters. These blocks are used to add audio in your creations.

stop all sounds::soundMutes all audio being played before this block was ran.

  • (1) sine
  • (2) square
  • (3) sawtooth
  • (4) triangle
change tempo by (20)Increases the tempo by a number of bpm. The tempo affects how long notes and rests are, but not sounds and frequencies.
set tempo to (60) bpmSets the tempo to number.
play frequency (440) Hz::soundSets the frequency for a sprite to a number. That sprite will play that frequency until it is changed again or stopped.
stop frequency::soundStops the frequency.
Reporters (3)
[duration v] of sound [ v]::reporter soundReturns a value of properties of a sound effect.
  • Name: Returns the name of the sound effect.
  • Duration: Returns a number of the length the sound effect.
  • Length: Returns a number of how many samples are there.
  • Number of Channels: Returns a number of how many channels were on the sound effect.
  • Sample Rate
  • Samples: Returns a list of all samples taken out of the sound effect.
Returns a new sound from the samples.
