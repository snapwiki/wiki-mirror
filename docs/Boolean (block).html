Boolean block
<true>
<false>
CategoryOperators
TypePredicate

The Boolean block is a predicate and an operators block. It reports a Boolean, which can be one of the two values: true and false.

Redundancy

It is redundant using the Boolean block on Boolean inputs, as the user could click on the empty Boolean slot to declare a value in that slot.

Use this......instead of that
set [video capture V] to <t>set [video capture V] to <<true>>

Also, checking if a Boolean is true using the () = () block is a missed opportunity to cut down on the amount of blocks. To check if a Boolean is false, wrap the Boolean in a not () block instead.

Use this......instead of that
<mouse down?><<mouse down?> = <<true>>>

Example Uses

Reasonable uses to use the Boolean block include but not limited to:

Passing a Boolean to an input that a Boolean would mean a completely separate thing:

set [dark mode V] to <<true>>
