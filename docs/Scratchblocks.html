Scratchblocks is a plugin for the Snap! Wiki and Forums maintained by the user blob8108 that allows script images to be constructed through text that is parsed and rendered. The plugin itself can be played around with on its homepage. Example:

define dance (speed)
set [dist v] to ((speed) * (distance to [mouse-pointer v]))
repeat until <(timer) > [10]>
  move (dist) steps
  say [Wow! Snap! scripts on the wiki!]
end
