A sprite has a position (x, y) and a direction. The X of the position is horizontal one, goes left to right. The Y of the position is vertical one, goes down to up. The direction (angle) is where the sprite points at. 90 is right, 270/-90 is left, 180 is down, and 0 is up.

Motion blocks uses and changes both the position and direction. The stage does not have a position and direction, so you cannot add motion blocks to the stage.

Direction Types

There are 3 types of direction that changes the way the sprite behaves.

  • All: The sprite can face all directions.
  • Sideways: The sprite can only face left (270) and right (90).
  • Disabled: Direction behavior is disabled.

The direction type only affects how the costume is oriented; the point blocks and direction reporter still operate normally value-wise.

The direction type can be changed with the set block in the variables category. In the dropdown menu, open the "my" submenu and select "rotation style". The direction type can be accessed with the my reporter in the sensing category. In the dropdown menu, select "rotation style". In both blocks, 0 is disabled (don't rotate), 1 is all (can rotate), and 2 is sideways (only face left/right). The UI has buttons to change the direction type; the current direction type button is lighter than the others.
