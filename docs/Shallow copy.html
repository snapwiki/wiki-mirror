See also: Deep copy

Shallow copies duplicate as little as possible. A shallow copy of a collection is a copy of the collection structure, not the elements. With a shallow copy, two collections now share the individual elements. A shallow copy of a list can be created by setting a variable to a list directly. For example, running set [original v] to (list [1] [2] [3] :: list) :: variable, and then set [shallow copy v] to (original) :: variable makes (shallow copy) a shallow copy of (original). This means that if you run set [original v] to (list [4] [5] [6] :: list) :: variable, (shallow copy) is also changed to (list [4] [5] [6] :: list).
