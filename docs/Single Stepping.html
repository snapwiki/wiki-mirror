Single Stepping is a form of Visible Stepping, but scripts are not automatically stepped through. The pause button turns into a step button. This is useful for debugging a program and it allows the programmer to study variables, current script positions, and other parts of the program before clicking the step button to move the program forward one step.

To turn on Single Stepping, turn on Visible Stepping (footprints icon at the top) and slide the speed slider all the way to the left.
