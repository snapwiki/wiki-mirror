Arcs is a library in Snap!. It is one of the several libraries with only one block until Snap! 10.0, the others are List comprehension, Replace Letters, Just Bars, Fancy Stage Writing[upcoming in 10.0] and Script Pic Costumes[upcoming in 10.0]. In this case, the only block turns the sprite while moving, revolving the sprite around a circle when ran continuously.
