if on edge, bounce is a command block and a motion block. It first checks if the sprite is touching the edge, and if it is, the sprite "bounces" off the edge by turning the sprite away.

if on edge, bounce::motion

Example Uses

Bouncing a sprite around the screen:

when green flag clicked::control
go to x: (0) y: (0)::motion
point in direction (45 v)::motion
forever
move (2) steps::motion
if on edge, bounce::motion
end

Workarounds

This default definition can be found on Snap! 10.0 with Blocks all the way on. Remove or delete the block immediately after the definition hat if you want the script to be in action.
