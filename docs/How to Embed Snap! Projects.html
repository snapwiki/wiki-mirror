You can embed Snap! projects in your website.

Embedding in HTML

Go to a project page, like this one.

Click the "Embed" button to get some HTML code to paste into your website.

You can toggle on and off various elements:

  • Project title
  • Project author
  • Edit button
  • Pause button

Embedding on this wiki

In source editing mode, add the following tag, with the project and user name:

<snap project="helloworld" user="techboyg5" />
