change y by ()
change y by (10)
CategoryMotion
TypeCommand

This block is used to move a Sprite on the y-axis up or down.

Usage

This block can only be used on Sprites. Attempting to use this block on the Stage will result in an error.

On Stage

To use this block on the Stage, you must use a tell block. tell [Sprite V] to ({change y by (10)} @addInput) @addInput


On a Sprite

To use this block on a Sprite, drag it into the development area. Inputting a positive number will make the Sprite move up. A negative number will do the inverse and make the Sprite move down.
