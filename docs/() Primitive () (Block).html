Upcoming Feature
This page is about an upcoming feature. It can be previewed in a developer version. It is possible that the behaviour may change in the future. For more information, see Snap! 10.0.

() primitive () is a command block that is used to give primitive blocks (including itself) its function in Snap! 10.0. This block must be connected directly to a definition hat, otherwise it will throw an error. There's a Boolean switch, switching it off will make the block do nothing. Scripts placed under it will not run, and if the dropdown is set to a block where the script will be capped, activating it will turn the block into a cap block. It is only visible if the Extension blocks in settings is enabled.
