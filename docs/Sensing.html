Sensing is the sixth block category, and it contains 4 commands, 14 reporters and 6 predicates. (3 as stage) These blocks are used to get data about the current state of Snap!

BlockFunctionOptions
Commands (4)
Opens a text box that the user can type in. The message is placed by the sprite (like say) if possible, and otherwise is placed above the text box. Because it is a command block and not a reporter, the output is always piped to answer.
Sets the timer to zero
Turns on or off the feature specified
  • Turbo Mode
  • flat line ends
  • log pen vectors
  • video capture
  • mirror video: Mirrors the video horizontally
