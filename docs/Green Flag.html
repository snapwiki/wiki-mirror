The green flag is a push button placed on the IDE by create control bar, which behaves like the "start of the project". The icon has a green flag. (lightning bolt if turbo mode is enabled) The colour is always #00c800. When clicked, the flag triggers all when flag clicked blocks. In Scratch, it first stops all scripts and deletes all clones.

Clicking the green flag can be simulated programmatically with the broadcast blocks two ways:

  • Shift-click the drop-down menu. Click the green flag icon that appears in the menu. The block now says green flag clicked. Note: this does not work if you make a message called green flag clicked.
  • Create a new message and call it __shout__go__. The receive hat block with that message in it is also triggered when the actual green flag is clicked.

Note: when you click the green flag, the (message::control) reporter does not change.
