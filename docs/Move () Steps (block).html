The move (10) steps block is a block in the Motion category that moves the sprite in the direction it is facing.

Examples

Making sprites follow each other or the mouse pointer:

when green flag clicked :: control
forever :: control
point towards [mouse-pointer v] :: motion
move (10) steps :: motion

Moving in a circle:

when green flag clicked :: control
forever :: control
turn cw (15) degrees :: motion
move (10) steps :: motion
