Pen is the fourth category of blocks, and it carries 13 blocks, 3 reporters, and 1 predicate. These blocks are used to manipulate the pen trails. Lines are drawn by moving while the pen is down.

stamp::penAdds the sprites current costume to the pen trails at its current position.
fill::penStarts a flood fill from the sprite's position using the sprite's pen colour.
write [Hello!] size (12)::penWrites the string given at the location of the sprite. Direction is preserved and the sprite is moved to the end. If pen is down, text will be underlined.

Reporters (3)
(pen [hue v]::pen)Reports the colour attribute specified.
  • hue
  • saturation
  • brightness
  • transparency
(pen trails::pen)Reports the pen trails as a costume. It will not be affected by further use of pen.
(pen vectors::pen)This block is only accessible in the (pen trails::pen) context menu.

Like the pen trails block, but reports the pen trail as a vector costume, but it only does anything if the Snap! option log pen vectors is enabled.
