Pen is the fourth category of blocks, and it carries 12 blocks, 2 reporters, and 1 predicate. These blocks are used to draw on the stage. Lines are drawn by moving while drawing mode is enabled.

BlockFunctionOptions
Commands (9)
clear :: penRemoves anything drawn on the stage.
pen down :: penStarts drawing. Disabled on the stage.
pen up :: penStops drawing. Disabled on the stage.
set pen color to [#911a44] :: penSets the colour to draw with.
change pen [hue v] by (10) :: pen Increases the attribute specified by a number
  • hue
  • saturation
  • brightness
  • transparency
set pen [hue v] to (50) :: penSets attribute to a numberSame options as previous.
change pen size by (1) :: penIncreases the pen width by the number. The pen is a circle.
set pen size to (1) :: penSets the pen width to a number.
stamp :: penMakes a copy of the sprite in the pen layer. That copy is part of pen trails, and cannot be modified.
fill :: penFlood fills all space near the sprite with that sprite's pen colour.
write [Hello!] size (12) :: penWrites the string given at the location of the sprite. Direction is preserved and the sprite is moved to the end. If pen is down, text will be underlined.
paste on [ v] :: pen Makes a copy of the sprite and places it on another sprite.Stage and sprite names
cut from [ v] :: penMakes a hole in the selected sprite with the shape of the current sprite.Stage and sprite names
