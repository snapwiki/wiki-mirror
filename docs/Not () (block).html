Not () is a block that reports the inverse Boolean. This means if true was put into the block it will report false, and false will report true.

not <>

Example Uses

Making sure the mouse is up before repeating the action

forever
if <mouse down?> {
change [value v] by (1)
wait until <not <mouse down?>>
}::control

Workarounds

One of the workarounds are:

<(Boolean) = <false> @delInput @addInput::operators>
