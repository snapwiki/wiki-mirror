current ()
(current [date V])
CategorySensing
TypeReporter

current () is a command block and a sensing block. It reports the current local date or time of the user. The result is not padded with leading zeros (7 in situations where it could display 07).

Options

  • year: The current four-digit year, like 2024.
  • month: The current month, expressed as the nth month of the year (not the month name like July), like 07.
  • date: The current day, like 15.
  • day of week: The current day of week.
  • hour: The current hour in 24-hour time, starting from 0 which is 12 AM, like 1.
  • minute: The current minute.
  • second: The current second.
  • time in milliseconds: Amount of milliseconds passed since ?. Use ((current [time in millseconds v]) mod (1000)) for the current millisecond.

Example Uses

The current () block can be handy for a particular group of projects, including but not limited to:

  • Clock, stopwatches and timers
  • Projects that use custom timers
