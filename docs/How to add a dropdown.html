Welcome! This is a tutorial on how to add a dropdown for a custom block!


First, add a custom block using the tutorial found in the article for custom blocks.


Next, right click on your new custom block and click "edit..."


Click on the label fragment you want to make a dropdown, or if you don't have one make a new one.

Now, click on the label fragment, turn it from a title text to a input name. Then, click on the triangle next to the selected "input name" setting.


Click on the cog, which will pop up with a new menu that should look like the image below

the "edit label fragment" menu.


Finally, select any other option than (none) to make a dropdown!

Or, just select "options..." to create custom options.
