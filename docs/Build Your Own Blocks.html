BYOB's project editor.

Build Your Own Blocks (BYOB) is a Scratch Modification of version 1.3 that introduces first class lists, sprites, and procedures. As the name suggests, users are given the tools of the ability to create blocks to use in their projects. This was in fact, the main feature of BYOB. BYOB is Snap!'s predecessor.
