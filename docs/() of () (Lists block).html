() of () is a reporter block and a lists block. It reports some overall properties of the supplied list.

([length V] of @list)

Options

  • length: Reports the amount of top-level items. For example, a list of numbers [6, 3, 7, 10] will report 4 as expected, but any higher-dimension lists will report the number of rows as shown in the table view.
  • rank: Reports the amount of dimensions. A simple, one dimensional list such as [1, 2, 3, 4] will report 1, but 2D lists (i.e. tables) report 2. A table of lists will report 3. This increases as lists are nested deeper.
  • dimensions: Reports the highest amount of items of each dimension.
  • flatten: Converts a multi-dimensional list into a single-dimension list.
  • columns: Reports the list with rows now acting as columns and vice versa.
  • uniques: Reports the list without duplicate items.
  • distribution: Creates a table with two columns. The first column consists of target items, and the second column consists of how many occurrences of that item.
  • sorted
  • shuffled: Randomizes the order of items.
  • reversed: Reverses the order of items so that the first item is the last and the last becomes first.
