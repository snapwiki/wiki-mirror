Pen is a Snap! feature used to draw on the stage.

Pen trails

Pen trails is the name for everything drawn by the pen.

The pen layer is the layer on the stage that displays the pen trails.

Usage

Each sprite has its own pen. The pen can be used in different ways to add to the pen trails.

The block puts the pen down. When the pen is down lines will be drawn whenever the sprite moves.

The block takes the pen back up, so it will stop drawing.

The block reports true if the pen is down and false if it's up.
