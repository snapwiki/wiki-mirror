Topic of the Month
Topic of the Month is a monthly event where Snappers are encouraged to make projects on a specific topic. It is known as TOTM for short. All projects made as part of this event can be added to that month's TOTM studio by any Snapper. The TOTM studio is featured on the Snap! homepage for the duration of the month...
