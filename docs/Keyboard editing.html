Keyboard editing is a way to edit scripts in the editor by just using the keyboard. Keyboard editing can be activated by clicking the keyboard icon in the top-right of the scripts area, shift-clicking the scripts area or typing Shift+Enter.

Keyboard Shortcuts

  • Up ↑: Navigates up a command block.
  • Down ↓: Navigates down a command block.
  • Left ←: Navigates to the previous input. If there's no more inputs for the block, moves up a command block.
  • Right →: Navigates to the next input. If there's no more inputs for the block, moves down a command block.
  • Backspace: Deletes a block. If used in variadic input slots, it removes an input.
  • Enter: If used in variadic input slots, it adds an input.
  • Space: Shows possible values for the current input slot. If it is a read-only dropdown, it shows the available options. If it is a normal (white) dropdown, it shows available options too while Enter allows typing.
  • Any other character that types something (for example, ç) opens the block search with the character on it.
