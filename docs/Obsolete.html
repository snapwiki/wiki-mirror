Obsolete!
(Obsolete!::red)

The Obsolete block (written 'obsolete!') is a red block that would be placed over any blocks that Snap! (and probably BYOB) cannot load, such as outdated, broken, or removed blocks. This block throws an error 'a custom block definition is missing'.

It also appears in custom block definitions which reference other custom blocks that were not exported with the project.
