Scratch is a visual programming language created by the Lifelong Kindergarten at the MIT Media Lab. Scratch uses blocks to generate code to make interactive content such as games, animations, music, stories, and much more. Scratch has a website where users can access the online project editor and view creations made by other users, or “Scratchers”. It is also a free coding software using blocks. Blockly is the program that Scratch's blocks are based on.

Scratch is still considered an inspiration for Snap!. Users have also described Snap! as a more modernised version of Scratch, despite this Snap! has evolved with more blocks than Scratch as well as continuing to have more technical features that Scratch does not have (e.g HTTP requests and JavaScript editing).

Modifications

Alternative Article on the Scratch Wiki

A Scratch Mod is a software (usually free) that uses Scratch's source code to create as such. Mods introduces features, such as more blocks, a different costume/backdrop and sound editor, new interface, and many more.

Snap!’s precessor, BYOB, was a modification of Scratch 1.4.
