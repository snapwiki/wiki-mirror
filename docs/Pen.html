the pen is a feature that all sprites have that allows them to draw on the stage.

the pen is controlled using blocks from the pen category.

the pen has several values associated with it, each sprite has its own pen, with its own unique values.

the pen has a colour, which is stored as an rgba object.

the pen can be up or down.

when up, the pen does nothing. when down, lines are drawn on the pen layer as the sprite moves.

the turtle costume changes colour to match the pen colour.
