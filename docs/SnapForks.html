SnapForks is a embed player that runs snap projects with the given Project Name and Author. When you give the given Project Name and Author, and press view SnapForks will run the found project

How to

How to use the player

See the two input boxes, enter the Project Name in the first box (eg. engine), enter the Author in the second box (eg. jens), then click view, now the project should show, if it doesn't check you input boxes.

How to get the embed code

First view a project (found at How to use the player), and the scroll to the bottom and see that textarea (eg. https://snap.berkeley.edu/embed?project=engine&user=jens&showTitle=true&showAuthor=true&editButton=true&pauseButton=true), copy the code then paste it into your website. Done!
