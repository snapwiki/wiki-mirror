Upcoming Feature
This page is about an upcoming feature. It can be previewed in a developer version. It is possible that the behaviour may change in the future. For more information, see Snap! 10.0.

3D Beetle Geometry is an upcoming library. It enables users to create 3D projects, similar to a Snap! modification titled Beetle Blocks.

Importing this library will automatically load the extension, consequently the "Open 3D Window" button will appear on top of the "Make a block" button. The 3D Window shows an interactive view of the current 3D scene, along with multiple options and a button to export:

  • Toggle visibillity of the beetle, grid, XYZ axes and extrusion base
  • Toggle wireframe
  • Toggle ghost mode, which makes extruded shapes transparent
  • Toggle first person view, positioning the view to the facing direction of the beetle
  • Toggle orthographic mode
  • Zoom to fit
  • Reset the camera
