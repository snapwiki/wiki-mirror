There are ways to check if a sprite is a clone.

Note that it only works for temporary clones which are created programmatically.

Using the My Block

(my [temporary? v]::sensing)

Use the my () block and set the input to "temporary?". Despite the shape, you can fit the block inside hexagonal inputs.

You can enclose it inside a not block for the opposite.

Manually

set [clone? v] to <true::operators>
create a clone of [myself v]::control
set [clone? v] to <false>

The sprite-local variables are separate for each instance among the clones, not just sprites.
