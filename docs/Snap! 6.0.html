Snap! 6.0 is the latest version of Snap!. It has been released on July 8, 2020.[1] It comes with several new features and big browser compatibility update (especially for Google Chrome).

Release

Snap! 6.0 was in alpha from May 6, 2020 to May 26, 2020.[2] Snap! 6.0 has been released on July 8, 2020 as mentioned in section above.

New Features

Hyperblocks

Snap! 6.0 allows users to conduct mathematical operations with lists (vectors) and lists of lists (matrices), which was not possible without mapping in earlier versions. If two lists are used, the resulting list has the same length as the shorter list.

Blockify

If a list contains nothing other than Booleans, numbers, texts, and other lists that match that criteria, you can use the Blockify feature. Right-click on the list, click blockify, and the mouse will pick up a newly generated list block with the list contents filled in.

New blocks

New List Blocks

The reporter that was in the List Utilities library is now a primitive block.

New Control Blocks

A new block was added to Snap! in the Control category called send [ v] to [ v] ::control.

Performance Upgrade

Snap! 6.0 used the new version of Morphic.JS, the JavaScript library written by Jens Moenig, the main developer of Snap! which draws the Snap! editor. This prevented a Google Chrome bug from happening. The amount of storage and loading time for projects also got reduced significantly.

User Interface Modernising Request

In the 6.0 beta (and formerly alpha) testing thread, the developers were requested to modernise the User Interface by adding special effect so that blocks turned into text after they were snapped. The developers later introduced Fade Blocks (see images below) to address this.
